About

This plugin vectrize whole image or select area.
It generate multiple color filled SVG pathes,and created as VectorLayer.

Recommend image size is smaller than 64px * 64px

This plug-in also need addict Library(Python)
https://github.com/mewwts/addict
Download it and into Pykrita folder(The directory of krita plug-ins)

This plug-in is python port of Imagetracer.js v1.2.6
https://github.com/jankovicsandras/imagetracerjs

User Tips

The result nothing

Assumed cause...
  • numberofcolors = 0
  • colorquantcycles = 0,it must be 1-3 value
  • Set line art mode, but strokewith is 0
  • Scaling is larger than canvas display area(The vector objects can selectable)

Too small result

Default tracing scale is 1 ( pixel: export vector size = 1:1),Both look similar
So,set scale parameter to 10 if you can see the difference of vector and pixel.

Get smooth curve from pixel

  • Error threshold for Line (ltres) value to smaller (0.05)
  • Error threshold Q-Spline (qtres) value to larger (1.0)

I want get grayscale result

  • Use colors (numberofcolors) parameter to less than 8 (1-7)

I want line art only,not want fill colors

  • Check Line art
  • Set Line Width to greater than 0
